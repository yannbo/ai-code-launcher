# Changelog

## v0.1.2

### Added

- 应用内 `Preferences...` 语言设置窗口
- 可手动切换 `中文` 和 `English`
- 可恢复默认：跟随系统语言
- GitHub README 改为中英双语

### Updated

- 最新 release 文案改为中英双语
- 安装包和磁盘镜像更新为新的语言设置版本

## v0.1.1

### Added

- 根据系统语言在中文和英文之间自动切换界面文案

### Updated

- 主窗口按钮、菜单、提示文案和弹窗都支持中英文
- 安装包和磁盘镜像更新为双语版本

## v0.1.0

首个公开版本。

### Added

- 原生 AppKit macOS 启动器
- 项目目录选择、最近使用、收藏和移除
- 支持导入某个父目录下的所有子文件夹
- Finder 中显示当前项目
- 可输入的启动命令下拉框
- 预置命令：`codex`、`claude code`、`opencode`
- 自动记住自定义命令
- `.app`、`.pkg`、`.dmg` 构建脚本
- 自定义应用图标生成脚本

### Notes

- 当前分发包未做签名和公证
- `.dmg` 采用稳定优先的标准拖拽安装方式
